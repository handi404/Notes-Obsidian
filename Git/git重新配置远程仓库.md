
执行如下命令：

```shell
cd 你要配置git远程仓库的文件夹 
# 移除远程仓库 
git remote rm origin 
# 添加远程仓库 
git remote add origin https://github.com/Saint9768/xxx.git 
# 查看远程仓库 
git remote -v
```
