// 队列 = FIFO 数据结构。先进先出 
// 设计用于在处理之前保存元素的集合 
// 线性数据结构 
// 入队 = Offer() 
// 出队 = poll() 

// 队列在哪里有用？ 
// 1. 键盘缓冲区（字母应按照按下的顺序出现在屏幕上） 
// 2. 打印机队列（打印作业应按顺序完成） 
// 3. 用于 LinkedLists、PriorityQueues、广度优先搜索
