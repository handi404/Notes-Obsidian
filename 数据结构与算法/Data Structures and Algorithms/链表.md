长话短说
小数据集：LinkedList = BAD 
大数据集 + 大量搜索：LinkedList = BAD 
大数据集+大量插入/删除：LinkedList = GOOD

// LinkedList = 节点分为两部分（数据 + 地址）
// 节点位于不连续的内存位置
// 元素使用指针链接

  //    优点？
  // 1. 动态数据结构（运行时分配所需内存）
  // 2. 节点的插入和删除很容易。O (1) 
  // 3. 无/低内存浪费

  // 缺点？
  // 1. 更大的内存使用（额外的指针）
  // 2. 没有元素的随机访问（没有索引[i]）
  // 3. 访问/搜索元素更耗时。O (n)

  // 使用？
  // 1. 实现栈/队列
  // 2. GPS 导航
  // 3. 音乐播放列表
