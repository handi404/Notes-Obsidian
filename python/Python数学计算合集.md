1. 「加法」：
    
    `def add(x, y):       
	    return x + y   `
    
2. 「减法」：
    
    `def subtract(x, y):
           return x - y   `
    
3. 「乘法」（已在先前的回答中给出）：
    
    `def multiply(x, y):
           return x * y   `
    
4. 「除法」：
    
    `def sectionide(x, y):
           return x / y   `
    
5. 「取模（求余数）」：
    
    `def modulo(x, y):
           return x % y   `
    
6. 「指数」：
    
    `def power(x, y):
           return x ** y   `
    
7. 「平方根」：
    
    `def square_root(x):
           return x ** 0.5   `
    
8. 「绝对值」：
    
    `def absolute_value(x):
           return abs(x)   `
    
9. 「自然对数」：
    
    `import math
    def natural_log(x):
	    return math.log(x)   `
    
10. 「以10为底的对数」：
    
    `def log_base_10(x):
           return math.log10(x)   `
    
11. 「正弦」：
    
    `import math   
    def sine(x):       
	    return math.sin(x)   `
    
12. 「余弦」：
    
    `import math   
    def cosine(x):
           return math.cos(x)   `
    
13. 「正切」：
    
    `import math   def tangent(x):       return math.tan(x)   `
    
14. 「反正弦」：
    
    `import math   def arcsine(x):       return math.asin(x)   `
    
15. 「反余弦」：
    
    `import math   def arccosine(x):       return math.acos(x)   `
    
16. 「反正切」：
    
    `import math   def arctangent(x):       return math.atan(x)   `
    
17. 「绝对值的幂次」：
    
    `def abs_power(x, y):       return abs(x) ** y   `
    
18. 「最大值」：
    
    `def max_value(x, y):       return max(x, y)   `
    
19. 「最小值」：
    
    `def min_value(x, y):       return min(x, y)   `
    
20. 「阶乘」
    
    `def factorial(n):       if n == 0:           return 1       else:           return n * factorial(n - 1)`
