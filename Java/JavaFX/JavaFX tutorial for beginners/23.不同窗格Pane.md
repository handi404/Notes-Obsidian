
## FlowPane

`FlowPane` 是JavaFX中的一个布局容器类，用于以流动的方式排列其子节点。它会按照子节点的添加顺序从左到右、从上到下地排列子节点，当一行的空间不足时，节点会自动换行到下一行继续排列。

### `FlowPane`的特点
- **自动换行**：当容器的宽度不足以容纳所有子节点时，`FlowPane`会自动换行，将多余的节点放到下一行。
- **水平和垂直间距**：你可以设置节点之间的水平和垂直间距（hgap 和 vgap）。
- **方向控制**：可以指定`FlowPane`的子节点排列方向（水平或垂直）。

### 创建一个简单的`FlowPane`示例

以下是一个基本的JavaFX应用程序，展示了如何使用`FlowPane`布局。

```java
import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.scene.layout.FlowPane;
import javafx.stage.Stage;

public class FlowPaneExample extends Application {

    @Override
    public void start(Stage primaryStage) {
        // 创建一个FlowPane
        FlowPane flowPane = new FlowPane();

        // 设置水平和垂直间距
        flowPane.setHgap(10);
        flowPane.setVgap(10);

        // 添加一些按钮作为示例节点
        for (int i = 1; i <= 10; i++) {
            Button button = new Button("Button " + i);
            flowPane.getChildren().add(button);
        }

        // 创建一个场景，并将FlowPane添加到场景中
        Scene scene = new Scene(flowPane, 300, 200);

        // 设置舞台（窗口）标题
        primaryStage.setTitle("FlowPane Example");

        // 将场景添加到舞台中
        primaryStage.setScene(scene);

        // 显示舞台
        primaryStage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}
```

### 主要属性和方法

- **`setHgap(double value)`**: 设置子节点之间的水平间距。
- **`setVgap(double value)`**: 设置子节点之间的垂直间距。
- **`setOrientation(Orientation value)`**: 设置子节点排列的方向，可以是`Orientation.HORIZONTAL`（默认）或`Orientation.VERTICAL`。
- **`getChildren()`**: 返回此`FlowPane`中的子节点的列表，可以用来添加或删除子节点。

### 使用示例
- 如果你将`FlowPane`的方向设置为`Orientation.VERTICAL`，它会先垂直排列节点，垂直空间不足时，才会换到下一列：

```java
flowPane.setOrientation(Orientation.VERTICAL);
```

### 适用场景

`FlowPane`非常适合用来创建那些不需要严格对齐的用户界面元素的布局，例如标签云、图片库缩略图、按钮组等。当容器的大小发生变化时，`FlowPane`可以自动调整子节点的布局，使其在不同窗口大小下都能合理排列。

通过 `FlowPane`，你可以快速布局一组组件，而不需要手动管理换行或重新排列子节点。


## GridPane

`GridPane`是JavaFX中的一个布局容器类，用于将其子节点放置在网格的行和列中。它非常适合用于创建表格、表单或其他需要精确对齐的布局。

### `GridPane`的特点

- **网格布局**：`GridPane`通过网格（由行和列组成）来布局其子节点。每个子节点可以被放置在特定的单元格（由行索引和列索引标识）中。
- **跨行和跨列**：子节点可以跨越多行或多列，允许更加灵活的布局。
- **对齐和间距**：可以设置单元格之间的水平和垂直间距，以及子节点在单元格内的对齐方式。
### 适用场景

`GridPane`非常适合用于创建需要精确对齐的布局，例如表单、计算器界面、棋盘或其他表格式布局。它的网格系统使得你可以轻松地在多个行和列中组织控件，并根据需要跨越行和列。
