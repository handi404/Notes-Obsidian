在 Java 中，嵌套数据结构同样非常常见，并且可以通过泛型和集合框架来实现。以下是一些常见的嵌套数据结构示例：

### 1. 嵌套列表

在 Java 中，嵌套列表可以通过 `List` 接口和 `ArrayList` 类来实现。例如：

```java
import java.util.ArrayList;
import java.util.List;

public class NestedListExample {
    public static void main(String[] args) {
        List<List<Integer>> nestedList = new ArrayList<>();

        // 创建子列表并添加到嵌套列表中
        List<Integer> sublist1 = new ArrayList<>();
        sublist1.add(1);
        sublist1.add(2);
        sublist1.add(3);

        List<Integer> sublist2 = new ArrayList<>();
        sublist2.add(4);
        sublist2.add(5);
        sublist2.add(6);

        nestedList.add(sublist1);
        nestedList.add(sublist2);

        // 访问嵌套列表中的元素
        int element = nestedList.get(1).get(2); // 访问第二个子列表中的第三个元素
        System.out.println("Element: " + element); // 输出 6
    }
}
```

### 2. 嵌套字典

Java 中没有直接的字典类型，但我们可以使用 `Map` 接口和 `HashMap` 类来实现嵌套字典。例如：

```java
import java.util.HashMap;
import java.util.Map;

public class NestedMapExample {
    public static void main(String[] args) {
        Map<String, Map<String, String>> nestedMap = new HashMap<>();

        // 创建子字典并添加到嵌套字典中
        Map<String, String> subMap1 = new HashMap<>();
        subMap1.put("subkey1", "value1");
        subMap1.put("subkey2", "value2");

        Map<String, String> subMap2 = new HashMap<>();
        subMap2.put("subkey3", "value3");
        subMap2.put("subkey4", "value4");

        nestedMap.put("key1", subMap1);
        nestedMap.put("key2", subMap2);

        // 访问嵌套字典中的值
        String value = nestedMap.get("key1").get("subkey2"); // 访问键'key1'对应的子字典中的键'subkey2'的值
        System.out.println("Value: " + value); // 输出 value2
    }
}
```

### 3. 嵌套集合

嵌套集合可以通过 `Set` 接口和 `HashSet` 类来实现。例如：

```java
import java.util.HashSet;
import java.util.Set;

public class NestedSetExample {
    public static void main(String[] args) {
        Set<Set<String>> nestedSet = new HashSet<>();

        // 创建子集合并添加到嵌套集合中
        Set<String> subset1 = new HashSet<>();
        subset1.add("A");
        subset1.add("B");

        Set<String> subset2 = new HashSet<>();
        subset2.add("C");
        subset2.add("D");

        nestedSet.add(subset1);
        nestedSet.add(subset2);

        // 访问嵌套集合中的元素
        for (Set<String> subset : nestedSet) {
            for (String element : subset) {
                System.out.println("Element: " + element);
            }
        }
    }
}
```

### 4. 嵌套自定义类

我们也可以创建嵌套自定义类来实现复杂的数据结构。例如：

```java
import java.util.ArrayList;
import java.util.List;

class Node {
    int value;
    List<Node> children;

    Node(int value) {
        this.value = value;
        this.children = new ArrayList<>();
    }

    void addChild(Node child) {
        children.add(child);
    }
}

public class NestedCustomClassExample {
    public static void main(String[] args) {
        Node root = new Node(1);

        Node child1 = new Node(2);
        Node child2 = new Node(3);

        root.addChild(child1);
        root.addChild(child2);

        Node grandChild = new Node(4);
        child1.addChild(grandChild);

        // 访问嵌套自定义类结构中的元素
        System.out.println("Root Value: " + root.value); // 输出 1
        System.out.println("Child Value: " + root.children.get(0).value); // 输出 2
        System.out.println("Grandchild Value: " + root.children.get(0).children.get(0).value); // 输出 4
    }
}
```

### 访问和操作嵌套数据结构

在 Java 中访问和操作嵌套数据结构需要多次调用 `get` 方法。例如：

```java
List<List<Integer>> nestedList = new ArrayList<>();
// 访问第二个子列表中的第三个元素
int element = nestedList.get(1).get(2);

Map<String, Map<String, String>> nestedMap = new HashMap<>();
// 访问键'key1'对应的子字典中的键'subkey2'的值
String value = nestedMap.get("key1").get("subkey2");
```

通过使用嵌套数据结构，可以更加灵活和有效地组织和管理复杂数据。
