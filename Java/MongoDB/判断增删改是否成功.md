在 Java 中与 MongoDB 进行增、删、改操作时，MongoDB 驱动程序通常会返回操作的结果对象，通过该结果对象，你可以判断操作是否成功。以下是一些常见操作及其结果处理方式：

### 1. 插入操作（Insert）

对于插入操作，使用 `InsertOneResult` 或 `InsertManyResult` 来检查插入是否成功。

```java
import com.mongodb.client.MongoClient;
import com.mongodb.client.MongoClients;
import com.mongodb.client.MongoCollection;
import com.mongodb.client.MongoDatabase;
import com.mongodb.client.result.InsertOneResult;
import org.bson.Document;

public class MongoDBInsertExample {
    public static void main(String[] args) {
        MongoClient mongoClient = MongoClients.create("mongodb://localhost:27017");
        MongoDatabase database = mongoClient.getDatabase("mydatabase");
        MongoCollection<Document> collection = database.getCollection("mycollection");

        // 创建要插入的文档
        Document doc = new Document("name", "John Doe")
                                .append("age", 30)
                                .append("city", "New York");

        // 插入文档
        InsertOneResult result = collection.insertOne(doc);

        // 检查插入操作是否成功
        if (result.getInsertedId() != null) {
            System.out.println("Document inserted successfully with ID: " + result.getInsertedId().asObjectId().getValue());
        } else {
            System.out.println("Insert operation failed.");
        }

        mongoClient.close();
    }
}
```

### 2. 更新操作（Update）

对于更新操作，使用 `UpdateResult` 来检查更新是否成功或影响了多少文档。

```java
import com.mongodb.client.MongoClient;
import com.mongodb.client.MongoClients;
import com.mongodb.client.MongoCollection;
import com.mongodb.client.MongoDatabase;
import com.mongodb.client.result.UpdateResult;
import org.bson.Document;

public class MongoDBUpdateExample {
    public static void main(String[] args) {
        MongoClient mongoClient = MongoClients.create("mongodb://localhost:27017");
        MongoDatabase database = mongoClient.getDatabase("mydatabase");
        MongoCollection<Document> collection = database.getCollection("mycollection");

        // 定义更新条件和更新内容
        Document filter = new Document("name", "John Doe");
        Document update = new Document("$set", new Document("age", 31));

        // 执行更新操作
        UpdateResult result = collection.updateOne(filter, update);

        // 检查更新操作是否成功
        if (result.getModifiedCount() > 0) {
            System.out.println("Document updated successfully.");
        } else if (result.getMatchedCount() > 0) {
            System.out.println("Document found, but no update was necessary.");
        } else {
            System.out.println("No document matched the query.");
        }

        mongoClient.close();
    }
}
```

### 3. 删除操作（Delete）

对于删除操作，使用 `DeleteResult` 来检查删除操作是否成功或影响了多少文档。

```java
import com.mongodb.client.MongoClient;
import com.mongodb.client.MongoClients;
import com.mongodb.client.MongoCollection;
import com.mongodb.client.MongoDatabase;
import com.mongodb.client.result.DeleteResult;
import org.bson.Document;

public class MongoDBDeleteExample {
    public static void main(String[] args) {
        MongoClient mongoClient = MongoClients.create("mongodb://localhost:27017");
        MongoDatabase database = mongoClient.getDatabase("mydatabase");
        MongoCollection<Document> collection = database.getCollection("mycollection");

        // 定义删除条件
        Document filter = new Document("name", "John Doe");

        // 执行删除操作
        DeleteResult result = collection.deleteOne(filter);

        // 检查删除操作是否成功
        if (result.getDeletedCount() > 0) {
            System.out.println("Document deleted successfully.");
        } else {
            System.out.println("No document matched the query.");
        }

        mongoClient.close();
    }
}
```

### 4. 插入多个文档（Insert Many）

对于批量插入操作，使用 `InsertManyResult` 来检查所有文档是否成功插入。

```java
import com.mongodb.client.MongoClient;
import com.mongodb.client.MongoClients;
import com.mongodb.client.MongoCollection;
import com.mongodb.client.MongoDatabase;
import com.mongodb.client.result.InsertManyResult;
import org.bson.Document;

import java.util.Arrays;
import java.util.List;

public class MongoDBInsertManyExample {
    public static void main(String[] args) {
        MongoClient mongoClient = MongoClients.create("mongodb://localhost:27017");
        MongoDatabase database = mongoClient.getDatabase("mydatabase");
        MongoCollection<Document> collection = database.getCollection("mycollection");

        // 创建要插入的文档列表
        List<Document> documents = Arrays.asList(
                new Document("name", "Alice").append("age", 25),
                new Document("name", "Bob").append("age", 28),
                new Document("name", "Charlie").append("age", 32)
        );

        // 插入多个文档
        InsertManyResult result = collection.insertMany(documents);

        // 检查插入操作是否成功
        if (result.getInsertedIds().size() == documents.size()) {
            System.out.println("All documents inserted successfully.");
        } else {
            System.out.println("Some documents may not have been inserted.");
        }

        mongoClient.close();
    }
}
```

### 总结

- **插入操作**：检查返回的 `InsertOneResult` 或 `InsertManyResult` 对象，确定文档是否成功插入。
- **更新操作**：使用 `UpdateResult` 的 `getModifiedCount` 方法判断更新是否成功。
- **删除操作**：使用 `DeleteResult` 的 `getDeletedCount` 方法判断删除是否成功。

通过这些结果对象，你可以在代码中处理成功或失败的情况，并采取相应的措施。
