## C 语言字符串函数概述

在 C 语言中，字符串实际上是使用空字符 `\0` 结尾的一维字符数组。C 语言标准库提供了一组丰富的字符串处理函数，这些函数通常包含在 `string.h` 头文件中。它们可以用于字符串的复制、连接、比较、查找、分割等操作。
